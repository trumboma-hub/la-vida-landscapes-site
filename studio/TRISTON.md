# How to post to your Journal

Hey Triston — here's the quick guide for posting to the Journal on lavidalandscapes.com.

## Logging in

1. Go to **https://lavida.sanity.studio**
2. Click **Sign in with Google** (or use the email/password Marty set up)
3. You're in.

Bookmark that page — it's where you'll always go.

## Writing a post

1. In the left sidebar, click **Journal posts**
2. Click the green **+** button (top right) → **Journal post**
3. Fill in the fields:

   - **Title** — the headline. Keep it natural, like you'd say it.
   - **URL slug** — auto-fills from the title. You can leave it alone.
   - **Category** — pick one: Seasonal / Case Study / Reflection / Project Note.
   - **Sub-tag** — short context. Examples: "Spring", "Lake Martin, AL", "Founder note". Optional.
   - **Hero image** — drag in a photo. Add **alt text** (one short sentence describing the image — required for accessibility).
   - **Excerpt** — one or two sentences. This is what people see on the Journal index page before they click in.
   - **Body** — the actual post. You can write paragraphs, add headings, bullets, quotes, links, and drop more images inline.
   - **Publish date** — defaults to right now. If you want to schedule it, change this date — the post stays hidden until that date passes.

4. Click **Publish** (top right, blue button).

The post is live on lavidalandscapes.com/blog.html within a few seconds. No website rebuild, no deploy, no waiting on Marty.

## Editing or unpublishing

- Open the post → make changes → **Publish** again.
- To take a post down: open it → click the three-dot menu next to Publish → **Unpublish**. It's hidden from the site immediately but stays in the Studio so you can republish later.

## Drafts

Anything you don't hit Publish on stays as a draft. Only you can see drafts. Pick it back up anytime.

## Image tips

- Photos straight from your phone work great. Sanity resizes them automatically for the site.
- Landscape (horizontal) photos look best as hero images — the index page crops to 16:10.
- Add **alt text** every time. It's how Google reads the image and how screen readers describe it.
